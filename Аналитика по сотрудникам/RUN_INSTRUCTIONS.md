# Инструкция по запуску

Для работы приложения нужно запустить **Бэкенд** и **Фронтенд** одновременно в двух разных терминалах.

### 1. Бэкенд (Python)
Откройте новый терминал и выполните:
```powershell
cd "c:\Users\dufubi\Desktop\Аналитика\Аналитика по сотрудникам\backend"
uvicorn main:app --reload
```
*Должно появиться сообщение: "Uvicorn running on http://127.0.0.1:8000"*

### 2. Фронтенд (React)
Откройте **второй** терминал и выполните:
```powershell
cd "c:\Users\dufubi\Desktop\Аналитика\Аналитика по сотрудникам\client"
npm run dev
```
*Нажмите на ссылку `Expected: http://localhost:5173` (Ctrl + Click), чтобы открыть приложение в браузере.*

---
**Если возникнут ошибки:**
- Убедитесь, что все зависимости установлены (`pip install -r requirements.txt` и `npm install`).
- Проверьте, что адрес сервера в `Dashboard.jsx` (`http://localhost:8000`) совпадает с запущенным uvicorn.
